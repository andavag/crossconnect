# CrossConnect 🌐

CrossConnect is a full-stack localized event discovery and hosting dashboard designed to help people find, organize, and register for community-driven gatherings. It was developed during a 24-hour sprint at the Hack for Humanity hackathon (Santa Clara University).

## ✨ Features

* **Event Discovery & Management:** A centralized dashboard for users to seamlessly create, discover, and sign up for local events.
* **Real-Time Map Geocoding:** Integration with OSMnx to provide accurate mapping and dynamic location data for community gatherings.
* **Robust Backend Architecture:** Custom-built backend designed to gracefully handle multiple concurrent users and real-time data processing.
* **Secure Data Handling:** Implementation of data sanitization and API rate-limiting to ensure reliable and secure communication between services.

## 🛠️ Tech Stack

* **Frontend:** React (Component-based UI)
* **Backend:** Flask (Python routing and API management)
* **Mapping/Geocoding:** OSMnx 
* **Other Tools:** Git (Version Control), REST APIs
